# brick-game
這是一款支援即時切換主題（霓虹、極光、像素、星雲）的打磚塊遊戲，視覺渲染、背景動畫、音效與合成音樂都會在切換時同步改變，用Antigravity 做的。本專案核心特色亮點：  四合一動態風格切換 (HUD 右上角控制)， 趣味道具與實體物理：   1.多重球 (Cyan)。   2.破壞雷射 (Pink)。   3.加寬擋板 (Green)。   4.磁吸擋板 (Yellow)。   5.底部防護罩 (Purple)。   6.彈性反彈控制。 豐富的關卡與漸進難度， 包含本地高分紀錄榜。This is a brick-breaking game that supports real-time theme switching (Neon, Aurora, Pixel, Nebula). Its visual rendering, background animations, sound effects and synthesized music all change simultaneously when switching themes. The game is developed with Antigravity. Core highlights of this project: Four-in-one dynamic style switching (controlled via the upper right corner of the HUD); Fun power-ups and realistic physics mechanics:
1. Multi-Ball (Cyan)
2. Destruction Laser (Pink)
3. Paddle Widening (Green)
4. Magnetic Paddle (Yellow)
5. Bottom Shield (Purple)
6. Bounce Elasticity Control

It features numerous levels with progressive difficulty and a local high score leaderboard.
